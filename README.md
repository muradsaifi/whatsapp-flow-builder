# WhatsApp Flow Builder

## Overview
This project is a Laravel and Vue.js application for building flows for a WhatsApp chatbot. It allows users to create, manage, and execute automated messaging workflows.

## Installation Instructions
1. Clone this repository: `git clone https://github.com/muradsaifi/whatsapp-flow-builder.git`
2. Navigate into the project directory: `cd whatsapp-flow-builder`
3. Install dependencies:
   - For Laravel: `composer install`
   - For Vue.js: `npm install`
4. Set up your `.env` file and run migrations: `php artisan migrate`
5. Launch the application: `php artisan serve`
